Название: Movie-collection

Цель проекта: Разработать полнофункциональное веб-приложение с использованием JavaScript для закрепления навыков работы с DOM, событиями, массивами объектов и локальным хранилищем.

Информация об авторе:

Имя и Фамилия: Амангельды Атаниязов.

Группа: 16 РПО/25/ Курс: 1.

Открыть репозиторий проекта на Github

Инструкция по использованию:

Добавление фильма: Заполните все поля формы и нажмите «Сохранить».

Удаление фильма: В карточке нажмите «Удалить фильм».

Отметка о просмотре: Нажмите «Отметка о просмотре» или поставьте галочку.
