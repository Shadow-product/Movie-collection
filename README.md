# 🎬 Movie-collection

## Описание

Movie-collection - это учебное полнофункциональное веб-приложение с использованием JavaScript для закрепления навыков работы с DOM, событиями, массивами объектов и локальным хранилищем.
Проект должен предоставлять собой интерактивный инструмент для коллекционирования, с понятным интерфейсом и сохранением данных между сессиями.
Пользователь может добавлять фильмы, отмечать просмотренные, удалять записи и переключать тему (светлая/тёмная).  
Интерфейс адаптивный: на мобильных устройствах работает бургер‑меню. На десктопах бургер-меню скрыт и на планшетах.

---

## Проект создан для практики работы с HTML, CSS и JavaScript:
- Добавление фильмов через форму,
- Отображение карточек на главной странице,
- Отметка «просмотрено/не просмотрено» с помощью чекбокса,
- Удаление фильмов,
- Переключение светлой и тёмной темы,
- Адаптивный дизайн с бургер‑меню для мобильных устройств.
- На десктопах бургер-меню скрыт и на планшетах.

---

## Возможности:
- 📌 Добавление фильмов с указанием названия, года, жанра, рейтинга, чекбокса и постера.
- ✅ Отметка просмотренных фильмов (чекбокс).
- 🗑️ Удаление фильмов из каталога.
- 🎨 Переключение темы (светлая/тёмная).
- 📱 Адаптивный интерфейс (бургер‑меню на телефонах).
- 💾 Сохранение состояния фильмов и темы в localStorage

---

## Технологии:
- **HTML5** — структура страниц.
- **CSS3** — адаптивный дизайн, переменные для тем.
- **JavaScript** (ES6 Modules) — логика приложения, работа с DOM, сохранение данных.
- **localStorage** — хранение состояния фильмов и темы.
